Summary

  • round(num,p) 函数：
    num保留p位小数，例如round(7253.86, 0) -> 7254，返回数值
  • length(s) 函数：
    返回字符串s的长度
  • left(s,p) 函数：
    从字符串s开头截取p个长度的子串，left(‘Hello’,4) -> ‘Hell’
  • case when 函数：
    相当于if函数，但是case when更加适用于多条件的判断，返回值就在语句then和else后，case可以在不同条件下返回不同的值；主要用于条件聚合，打标签进行分类聚合的场景
